# engine

Minimal clojure 2D game engine for desktop.
Based on [libGDX](http://libgdx.badlogicgames.com/).

# Features

* images
* spritesheets
* animation
* shape drawing
* includes a pixel font
* input handling
* tilemap (tmx) support with lightning/shadow
* scaled viewport for pixelart

# Demo

```
lein -m engine.repl
```
